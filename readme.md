#  Demo EF

## Nuget packages necessaires pour EFCore : 
- Microsoft.EntityFrameworkCore
- Microsoft.EntityFrameworkCore.SqlServer
- Microsoft.EntityFrameworkCore.Tools
- Microsoft.EntityFrameworkCore.Design

## Mise en place de la classe « DbContext »
https://learn.microsoft.com/en-us/ef/core/dbcontext-configuration/

## Les commandes pour intéragir avec Entity Framework Core
### Dans le terminal du package manager
Obtenir la documentation de EFCore
```
get-help EntityFrameworkCore
```

La liste des commandes de EFCore
```
Add-Migration
Bundle-Migration
Drop-Database
Get-DbContext
Get-Migration
Optimize-DbContext
Remove-Migration
Scaffold-DbContext
Script-DbContext
Script-Migration
Update-Database
```
